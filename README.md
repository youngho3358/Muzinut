## 🙌🏻 프로젝트 소개
+ Muzinut은 뮤지션들이 서로의 음원을 업로드하여 공유하고 소통할 수 있는 사이트입니다.
+ 아티스트는 자신의 앨범(음원) 작업물을 업로드할 수 있습니다.
+ 게시판을 통해 아티스트와 리스너 간의 소통을 지원합니다.
+ 리스너는 검색을 통해 사용자는 업로드되어 있는 음원을 스트리밍하여 청취할 수 있습니다.
